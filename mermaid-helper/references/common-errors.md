# 常见错误

## 代码块格式错误

| 错误类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| 未使用代码块 | 直接输出代码 | ` ```mermaid ... ``` ` |
| 缺少语言标识 | ` ``` ` | ` ```mermaid ` |
| 未关闭代码块 | ` ```mermaid ... ` | ` ```mermaid ... ``` ` |

## 节点 ID 错误

| 错误类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| ID 含连字符 | `my-node` | `my_node` |
| ID 含空格 | `my node` | `my_node` |
| ID 重复 | 两个 `A` 节点 | `A1`, `A2` |

## 语法声明错误

| 错误类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| 缺少图表类型 | 直接写节点 | 先声明类型 |
| 类型拼写错误 | `flowshart` | `flowchart` |

## 结构闭合错误

| 错误类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| 子图未闭合 | `subgraph` 无 `end` | 添加 `end` |
| 循环未闭合 | `loop` 无 `end` | 添加 `end` |
| 条件未闭合 | `alt` 无 `end` | 添加 `end` |

## 箭头语法错误

| 错误类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| 时序图箭头错误 | `A->B` | `A->>B` |
| 流程图箭头错误 | `A -> B` | `A --> B` |

## 中文标签处理

部分情况下中文标签需要用引号包裹：

```
A["中文标签"]
```

## 注释语法

所有图表类型统一使用 `%%` 作为注释符号：

```
%% 这是注释
```
